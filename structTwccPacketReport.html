Detailed Description

Per-packet report from TWCC feedback for congestion control algorithms.

This structure provides detailed information about individual packets from TWCC (Transport-Wide Congestion Control) feedback. Applications can use this to implement congestion control algorithms like GCC (Google Congestion Control).
